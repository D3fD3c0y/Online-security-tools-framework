# Online Security Tools Framework

A static GitHub Pages website that provides a searchable, collapsible framework of online security resources.

Page hosted at https://d3fd3c0y.github.io/Online-security-tools-framework/?sel=Free+Online+Security+Services

## Overview

This project presents security resources in a structured tree view designed for quick navigation and reference.  
It is published as a static site and includes search, filtering, inline details, and lightweight UI enhancements for easier day-to-day use.

## Features

- **Collapsible tree view** for browsing categories and resources
- **Search and filtering** for faster discovery
- **Inline item details** directly in the tree
- **Light / dark theme toggle**
- **Copy/share button** for quick URL sharing
- **Build version and timestamp display**
- **Automated deployment** with GitHub Actions
